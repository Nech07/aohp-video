# AOHP Launch Video

Remotion launch video for [AOHP](https://github.com/aohp-os/aohp) — the Android Open Harness Project, an agent-native open fork of Android.

Built with the [Launchpad](https://github.com/trycua/launchpad) monorepo template.

## Quick start

```bash
cd launchpad
pnpm install
pnpm --filter @launchpad/Aohp remotion   # preview in Remotion Studio
pnpm --filter @launchpad/Aohp render     # render to out/video.mp4
```

See `launchpad/videos/aohp/SCRIPT.md` for the full scene breakdown and `launchpad/videos/aohp/README.md` for project details.
